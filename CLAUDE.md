# 技術士第一次試験 過去問解説サイト

## プロジェクト概要

技術士第一次試験（森林部門・専門科目）の過去問解説をまとめたWebサイト。
受験者が検索で流入し、問題ごとに答え合わせができる構成。

## 決定事項

### コンテンツ
- 対象：R3〜R7年度 森林部門 専門科目（5年分、各35問）
- 問題文：掲載あり（出所：日本技術士会 公式PDFを明示）
- 公開方針：全問無料、広告なし
- 解説フォーマット：下記参照

### 解説フォーマット
```
【正答】〇

【解説】
1〜3文。核心だけ。

【参考】
信頼できる情報源のURL。なければ省略。
```

### 収益化・連携
- 収益化：当面なし（将来クイズサイトへの導線）
- note（ほおのき）：告知・導線として活用
- 将来：技術士試験クイズサイトと統合予定

## 解説制作フロー

### ステップ1：テキスト化
- MacのプレビューアプリでPDFの文字認識→コピー→`r*_question.txt` に保存
- Claude Codeに指示：「r*_question.txt と answer.txt を読んでCSVを更新して」
- CSVを目視確認・誤認識を修正（詳細な作業手順・OCRミスパターンはmemoryを参照）

### CSVフォーマット
```csv
"year","number","question","answer","choice1","choice2","choice3","choice4","choice5"
"2024","1","問題文...",4,"①選択肢1","②選択肢2","③選択肢3","④選択肢4","⑤選択肢5"
```

### CSVの現状
- R3〜R7：全年度作成済み（35問）

### ステップ2：解説生成
- Claude Codeに指示：「CSVを読んで全問の解説MDを生成して」
- 1問ずつAPIを呼び出して処理・保存（サブエージェント方式）
- 解説MDを確認・必要なら修正（内容の精査はチャットで相談）

### 解説MDの保存先
```
content/
└── r6/
    ├── r6_III_01.md
    ├── r6_III_02.md
    └── ...
```

### 解説MDのフォーマット
```markdown
---
year: 2024
number: 1
question: "問題文..."
answer: 4
---

【正答】④

【解説】
...

【参考】
...
```

## 注意事項

- 問題文の著作権は日本技術士会に帰属。出所を明示して掲載。指摘があれば即対応。
- 解説はClaude AIで生成後、帆がレビューして掲載。
